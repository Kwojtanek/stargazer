Api description

The API endpoint is available under /searchAPI?format=json URL.
Via GET params you can filter the list.
Example:
/searchAPI?format=json&const=and&&max_mag=10
For some params it's posible to provide list of values. To do that separate them with comas.
Example:
/searchAPI?format=json&const=and,her,cas&&max_mag=10
